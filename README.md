# Fart_MiddleFinger_Ring

A Web-Game Artwork.

### How to Play? 
Remember our childhood toy, the water ring game?

1.Click the fart buttons to emit fart bubbles.  
2. Use the bubbles to put the floating rings on the middle finger.
